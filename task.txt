ЗАДАНИЕ 1

Написать API endpoint, куда клиент может закинуть файл, название которого будет в формате "DDMMYYYY_HHMMSS.format"
(пример, "15062022_110228.mp3". Название файла обозначает дату аудиозаписи).
Сервер должен спарсить название файла в дату и время аудиозаписи, и внести в БД данные о файле в колонки "fileName"(название файла),
"fileDate"(дату аудиозаписи), "filePath"(путь к файлу). Использовать FrameWork SPRING.

==============================

ЗАДАНИЕ 2

Создать  соединение для связи с клиентской частью. При поступлении команды "fileInfo" от клиента,
сервер должен отправить данные о файлах из базы данных из первого задания.
Пример: клиент отправляет запрос в виде {command: "fileInfo", fileName: "15062022_110228.mp3" },
сервер находит данные о файле из БД и кидает информацию клиенту в виде { fileName: "15062022_110228.mp3",
fileDate: (дата и время файла в читабельном формате), filePath: (путь к файлу) }.


А вот задание

Создать WebSocket соединение для связи с клиентской частью.
 При запросе { "command": "addLog", "content": "someText" }
 сервер должен сохранить значение "content" в БД. При запросе
 { "command": "logs" }, сервер должен передать массив,
  в котором находятся все предыдущие запросы от клиента.
P.S.: Клиентская часть долж.на быть внутри Java Spring Boot проекта.
